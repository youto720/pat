### プロンプト（そのままコピペ）:

```
React + TypeScript + Vite でゲームアプリ「pat」を新規作成してください。

## やること（Phase 1）

1. `npm create vite@latest pat -- --template react-ts` でプロジェクト作成
2. Tailwind CSS をセットアップ
3. 以下のゲームを実装してください

## ゲーム仕様

グリッド状のパネルを指（またはマウス）でなぞる一筆書きパズルゲームです。

### 初期グリッド
- 5列 × 7行のグリッド（後で拡張可能）
- マス1つのサイズは画面幅に応じてレスポンシブ調整

### スタート（S）とゴール（G）
- スタートマス: オレンジ色 (#F4845F)、グリッド左上付近にランダム配置
- ゴールマス: ピンク色 (#E87070)、グリッド中央付近にランダム配置
- 毎回ゴールするたびにS/Gの位置はランダムに変わる

### 操作
- スタートマスを押した瞬間になぞり開始
- 指（マウス）を離さず、隣接マスへ移動するとそのマスが緑 (#4CAF7D) に変わる
- ゴールマスに到達したらゲームクリア
- 途中で指を離す / 同じマスを再度通る → リセット（全マスが白に戻る）
- 斜め移動は不可（上下左右のみ）

### スコア
- 1マスなぞるごとに +10点
- 画面上部に累積スコアを表示
- ゴール後は3秒後に次のラウンドへ自動移行

### 音（Web Audio API で合成・ファイル不要）
- マスをなぞるたびに短い「ポン」音（音程が少し上がっていく）
- ゴール時に明るい3音のファンファーレ
- リセット時に低いシュッという音

### ゴールエフェクト
- 「GOAL!」テキストが中央に1.5秒表示
- パーティクルアニメーション（CSS のみでOK）
- スコア加算の浮き上がりアニメーション

### UI
- ヘッダー: 左にハンバーガーアイコン、中央に「slide」（太字）、右にスコア
- グリッドは画面中央、角丸の外枠
- 背景: 白 (#FFFFFF)
- フォント: Google Fonts の Nunito を使用
- グリッドの線: 薄いグレー (#E0E0E0)

### レベルアップ
- ゴール3回ごとにグリッドが1列拡大（最大8列）
- ゴール5回ごとにグリッドが1行拡大（最大12行）

### 技術的な注意点
- touchstart / touchmove / touchend を使い、passive: false でスクロール防止
- マウスイベント (mousedown / mousemove / mouseup) も同様に実装
- user-select: none でテキスト選択防止
- iOS Safari の -webkit-tap-highlight-color: transparent を設定
```
